To get more Language voices:
Windows → Settings → Time & Language → Language → (Add a Language) → Install "Speech" pack.

macOS → System Settings → Accessibility → Spoken Content → System Voice → (Add a Language voices).

Android → Settings → Language & Input → Text-to-Speech Output → (Install a Language).

ChromeOS → Settings → Advanced → Accessibility → Text-to-Speech → Manage voices.

Then reopen the browser.
